# twitter clone project

This simple project is developed in ReactJS via vite to showcase my knowledge in React and React Router. Post can be added and viewed.

This dummy backend server must be downloaded and installed for this project to work [twitter-clone-dummy-backend](https://github.com/papiyinks/twitter-clone-dummy-backend)

## Installation

You can fork and clone the project and run `npm install` to install the external dependencies.

## Helpful commands

You have the following CLI commands available:

- `npm run dev` - Runs the project and serves it on http://localhost:5173

- `npm run build` - Builds the project for production deployment
